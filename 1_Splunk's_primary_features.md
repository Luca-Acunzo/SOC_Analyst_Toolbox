# Index
Contains machine data from sources such as **servers, network devices and web applications**. The data is **processed** and **labeled** with a **sourcetype**. That sourcetype is then used to break the data into **individual events**.
# Splunk Search bar
To find events that with specific values. You can use the Splunk search language to **analyze** them and **generate statistics**.
# Report
Search results can be saved as reports, which can provide continued insights into your data. 
# Dashboard panels
You can use Report to power dashboard panels

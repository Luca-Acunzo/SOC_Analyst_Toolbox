# Search & Reporting app
Default interface for **searching** and **analyzing** data.
-  **Search bar**
- **Time range picker** (Intervallo temporale)
#### Data Summary Button (Sintesi dei dati)
- **Hosts**: Hostname, IP Address, fully qualified domain name of the machine from which the event originates
- **Sources**: The file or directory path, network port or script from which the event originated
- **Sourcetypes**: Classification of your data

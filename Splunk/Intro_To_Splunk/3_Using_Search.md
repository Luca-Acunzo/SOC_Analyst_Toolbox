# Search Job (Processo)
In the the search bar you type **failed** and set the **time range.**

![failed](Images/Splunk_Failed.png)

#### Visualization (visualizzazione)
Creating statistics and visualizations are called transforming commands.
#### Job (Processo)
- Inspect Job (Ispeziona)
- Delete Job (Elimina)
#### Share Job
By default, a search job will remain active for **10 minutes**.
Shared search jobs remain active for **7 days**.
#### Three search modes
- **Fast Mode**: Field discovery is disabled.
- **Smart Mode**: Based on the type of search you are running.
- **Verbose Mode**: Returns as much field and event data as possible.
